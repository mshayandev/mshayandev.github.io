WeatherNinja - Smart Weather Monitoring Station

Project information

  • Category: IoT Monitoring
  • Tech: Python, Firebase, Raspberry Pi 4B, DHT11, BMP280, YL-83
  • Award: Winner, Global IoT Solution Development (APTECH 2024)
  • Project URL: GitHub Repository

Designed a smart monitoring station with five sensors tracking temperature, humidity, pressure, rain, and light. The system achieved 99.9% data capture uptime and delivered extreme-weather alerts within 5 seconds through SMS, replacing manual logging with a reliable automated pipeline.
