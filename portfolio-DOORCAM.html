DoorCam - Smart IoT Security System

Project information

  • Category: IoT Security
  • Tech: Python, OpenCV, Flask, Firebase, Twilio, Raspberry Pi 4B
  • Award: 2nd Runner-Up, International IoT Ignite 2025 (APTECH)
  • Project URL: GitHub Repository

Built a facial-recognition based smart security system with 97% recognition accuracy across 50+ enrolled users and sub-2s identification time. Combined PIR, ultrasonic, and LDR sensors using multi-sensor fusion to reduce false positives by 80%, and enabled instant incident response with Twilio SMS and push alerts within 3 seconds.
